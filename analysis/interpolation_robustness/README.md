# Interpolation robustness analysis

These scripts compare metabolite resampling results generated by three methods:

1. Area-overlap weighted resampling (primary conservative method)
2. Nearest-neighbour resampling
3. Inverse-distance weighted resampling (IDW, k=4, power=2)

The analysis summarizes method robustness at the cluster level and across all cluster pairs using metabolite abundance correlations, cluster similarity correlations, log2 fold-change concordance, direction agreement and top-feature overlap.

## Expected inputs

- Resampled RNA-bin x metabolite matrices for each method.
- RNA-bin metadata with `bin_id`, `section_id` and `cluster` columns.

## Outputs

- Cluster-level average metabolite matrices.
- Method-level Spearman correlation summaries.
- Cluster-pair log2FC concordance tables.
- Top metabolite overlap summaries.
- PDF/PNG plots and Excel summary tables.
