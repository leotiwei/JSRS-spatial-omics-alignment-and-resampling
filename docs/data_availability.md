# Data availability and reproducibility notes

This repository provides custom code, workflow documentation and small demonstration inputs. Full raw human embryo images and full omics matrices are not included here.

The code is designed to run from processed inputs:

- transcriptomic spatial coordinates;
- registered or unregistered MSI coordinates;
- MSI feature intensity matrices;
- cluster metadata;
- curated reference mapping tables where required.

Example files under `example_data` are included to demonstrate the expected input formats and to allow basic code execution checks.
