# Input and output formats

## RNA coordinate table

Required columns:

- `x`: x coordinate in the RNA reference coordinate system.
- `y`: y coordinate in the RNA reference coordinate system.

Optional identifier columns:

- `bin_id`, `_index`, `barcode` or `Barcode`.

If no identifier column is present, bin IDs are generated by row order.

## MSI coordinate table

Required columns:

- `x`: registered x coordinate.
- `y`: registered y coordinate.

Optional identifier columns:

- `Spot_index`, `spot_id`, `msi_id`, `pixel_id` or `index`.

## MSI feature matrix

The MSI feature matrix should be spot x feature. The first column may contain spot IDs. Feature columns are metabolite or m/z feature names. The code also attempts to detect and transpose feature x spot matrices when spot IDs are present as column names.

## Resampled output

The resampled output is a CSV table with:

- `bin_id`
- `x`
- `y`
- one column per MSI feature

## Cluster metadata for robustness analysis

Required columns:

- `bin_id`
- `section_id`
- `cluster`
